PostCSS Colorblind

If the style of this page is a little different from the others, well, it is not a bug!

The style used within this page is generated using a PostCSS plugin called postcss-colorbind. It allows you to emulate how a colorblind person would see the page.

Caring about colors within your website is a huge responsability because it'll interfere directly on how your users will absorb the content you're offering.

Chrome Extensions

We can list a lot of cool Chrome extensions that helps us in our development journey. And, for colors, we can count on those beautiful extensions:

  • RGBlind
  • Dalton
  • Colorblinding
  • Colorblindly

As JS frameworks, Chrome extensions are countless and I bet that you may know another one that fits the job.

Caring about colors isn't a developer-exclusive subject - this must be an everyone's subject. From your team's leader, through your team's designer and your team's developers; if you work alone, you must endorse this subject and start caring about it.

Semantic Headings Example

This heading tag above is the most important in the page because it is an h1; it is the "page title", or the "content title".

Good case

The main point using heading tags is to place them in correctly order. It doesn't mean that you need to put an h4 right after an h3* - it means that you need to put the tags following the text order.

Bad case

The title above is incorrect because it is a second degree title and it is inside an h5 - visually it is stylized like an h2, but semanticly it is wrong and out of place.

* the h1 should be unique within your page; the usage of h2 to h6 must be in chronologically descending order.

But, imagine the following example:

  1. You have an article markup;
  2. The chapters are defined using h2, inside a section or even a div;
  3. Inside the chapters, you'll have subtitles;
  4. Those subtitles can be marked up with h3, h4;
  5. They're in chronologically descending order, even though you'll have an h2 for the next chapter after an h4.

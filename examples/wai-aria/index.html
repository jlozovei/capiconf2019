WAI-ARIA

What is it?

WAI-ARIA is a W3C specification that allows developers to apply a set of additional attributes for HTML tags, providing semantical and improving accessibility.

ARIA extends for Accessible Rich Internet Applications.

Basically, the WAI-ARIA attributes are divided between three main groups:

  1. Roles
  2. Properties
  3. States

How to?

Take the following example:

  1. You have a navigation bar;
  2. You can code it with an nav or with a div
  3. If you choose to create it with the nav, you're already semantic;
  4. But, if you choose to create it with a div, you'll need to use WAI-ARIA attributes to give this element a semantic meaning (which the nav already has);
  5. In this case, you could use a role attribute within your div;
  6. And, if this navigation is has some toggle effect, you can tell the user whether it is opened or closed with a aria-expanded attribute

Following it, you can create/modify the element's semantic meaning and its behavior when the user is using the keyboard to navigate.
